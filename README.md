# simulacionTrafico
Simulación de cómo se puede generar tráfico a pesar de no tener signos PARE o semáforos
Inspirado en experimentos controlados del estudio de tráfico (e.g Sugiyama Y, 2008) 
Se simula una sola calle circular sin semáforos o signos PARE, se puede observar que igualmente se genera congestión debido al retraso en tiempo de reacción humano(simulado)
Se crean olas emergentes de tráfico 

Cómo correr:

```bash
git clone https://github.com/YOUR_USERNAME/simulacionTrafico.git
cd simulacionTrafico
pip install -r requirements.txt
python simtraf.py
